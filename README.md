# Bikes Rental

## Configuration

Step 1: we need to add .env file in `server` folder <br />
Step 2: `cd client` <br />
Step 3: `npm install` <br />
Step 4: `cd ..` <br />
Step 5: `cd server` <br />
Step 6: `npm install`<br />
Step 7: `npm run dev`<br />
